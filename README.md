## Kraken - Borsuk 2

[![Built with open-wc recommendations](https://img.shields.io/badge/built%20with-open--wc-blue.svg)](https://github.com/open-wc)

## Quickstart

To get started:

```sh
npm init @open-wc
# niezbędny node w wersji > 10.16 & npm > 6
```
